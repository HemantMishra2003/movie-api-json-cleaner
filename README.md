# Movie_Json_Cleaner

 Movie API JSON Cleaner

This project fetches movie data from the OMDb API, extracts key details such as ratings, and converts the raw JSON response into a clean, structured DataFrame using pandas. It demonstrates how to work with public APIs and handle JSON data efficiently in Python.

 Features

Fetch movie data directly from the OMDb API
Convert complex JSON data into a clean tabular format (DataFrame)
Extract and display key fields such as ratings, title, year, genre, and more
Option to save cleaned data into a CSV file for later analysis
